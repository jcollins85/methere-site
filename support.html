Frequently Asked Questions

  • How do I back up my data?
    Use Settings → Export / Backup to save a file to the Files app or iCloud Drive.
  • How do I restore my data?
    Use Settings → Import and select a previously saved file.
  • How do proximity reminders work?
    If enabled, MetHere can notify you when you are near saved venues using on-device location features.
